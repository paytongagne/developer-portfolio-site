# Developer Portfolio Site

A lightweight personal portfolio website for presenting software projects, technical skills, and engineering experience in a clean, workplace-ready format.

This repository began as an early GitHub Desktop practice repo and is now being rebuilt into a real portfolio project. The goal is to turn an older repository into a polished front-facing project that supports job applications and gives recruiters a quick way to understand my work.

## Project Goals

- Present software, embedded systems, and cloud-connected application projects
- Organize technical skills by category
- Provide quick links to GitHub repositories and documentation
- Create a simple static site that can later be deployed with GitHub Pages
- Keep the design clean, readable, and recruiter-friendly

## Current Features

- Static `index.html` portfolio starter page
- Project metadata stored in `projects.json`
- Site planning documentation in `docs/site-plan.md`
- Recruiter-focused project positioning

## Featured Project Categories

- Embedded telemetry and backend data processing
- Python automation and Linux workflows
- Algorithms and data structures
- SQL-backed applications
- Configuration management and system administration

## Tech Stack

- HTML
- CSS planned
- JavaScript planned
- JSON project metadata
- GitHub Pages planned

## Repository Structure

```text

docs/site-plan.md   Design and content plan
```
