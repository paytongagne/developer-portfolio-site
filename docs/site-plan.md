# Site Plan

This portfolio site is designed to give recruiters a fast overview of my technical direction and project work.

## Primary audience

- Software engineering recruiters
- Technical hiring managers
- Entry-level and early-career engineering teams
- Embedded systems, IT, and backend-focused roles

## Main sections

1. Hero section with name, headline, and technical focus
2. Featured projects with concise summaries
3. Skills grouped by category
4. Experience highlights
5. Contact links

## Project positioning

The site should prioritize projects that connect to workplace skills:

- Embedded telemetry and backend data handling
- Python automation and Linux workflows
- Algorithms and data structures
- Web/database CRUD fundamentals
- Configuration management and system administration
